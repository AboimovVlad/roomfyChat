# Roomfy Chat 🤖

**Roomfy Chat** — телеграм‑бот для анонімного спілкування та ігор, реалізований на .NET Core з сучасною мікросервісною архітектурою та наступними технологіями:

* **RabbitMQ** — для асинхронного обміну повідомленнями між ботом та ігровими сервісами
* **Redis** — для кешування, зберігання сесій та ігрових станів (Tic‑Tac‑Toe)
* **SQLite** — легка основа для збереження користувачів та статистики
* **.NET Core** — телеграм‑бот та API, реалізовані на сучасному крос‑платформеному фреймворку

---

## 🚀 Функціональні можливості

* Реєстрація користувача через Telegram
* Анонімний чат з підтримкою всіх типів повідомлень Telegram (текст, фото, гіфки, стікери, відео)
* Ігра **Tic-Tac-Toe** через мікросервіс з RabbitMQ та Redis
* Тестування інтеграційними сценаріями

---

## 🚧 Запуск

```bash
git clone https://github.com/AboimovVlad/roomfyChat.git
cd roomfyChat
dotnet run --project RoomfyChat.Bot
```

* Redis та RabbitMQ повинні бути запущені локально
* SQLite створюється автоматично або вказується в appsettings.json
